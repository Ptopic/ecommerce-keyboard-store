<h1>How to run app? </h1>

<h2>Backend: </h2>

<p> Run Api </p>

```
cd backend
npm run start
```

<p> Run stripe payment webhook </p>

```
cd backend
stripe listen --forward-to localhost:3001/api/checkout/webhook
```

<h2>Frontend: </h2>

<p>Run Frontend </p>

```
cd frontend
npm run start
```

<h2>Admin Panel: </h2>

<p>Run Admin Panel </p>

```
cd admin
npm run start
```
